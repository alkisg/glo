# PSI
This is a Work-in-Progress ISO 7185 Pascal interpreter I am building in order to get confident with lexers, parsers, interpreters and all that good stuff. All PRs are wellcome!

### Currently supports:
* Data types
  * Integer
  * Real
  * Boolean
  * Char
* Variables
* Procedures
* Flow Control
  * If
  * Else If
  * Else
* Number Operators
  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (div)
  * Integer Division (/)
  * Modulus (mod)
* Logical Operators
  * AND
  * OR
  * NOT
* Comparison Operators
  * Equals (=)
  * Not Equals (<>)
  * Less than (<)
  * Less or Equal than (<=)
  * Greater than (>)
  * Greater or Equal than (>=)
